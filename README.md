El programa lee ciertos archivos CSV que contienen datos relacionados con cobertura móvil. Los CSV son obtenidos por el scanner y contienen columnas como GlobalCellID, frecuencia, PCI, latitud, longitud, entre otros.
Se limpian y normalizan los datos del CSV y se generan reportes en excel con los datos procesados. 
Adicionalmente, se generar archivos KMZ y PNG para los puntos filtrados y considerados, por diferentes parámetros:
-Banda de frecuencia
-GlobalCellId
-MCC_MNC
-Potencia RSRP
-Calidad RSRQ
